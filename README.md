### 🕳️ [Blackhole Desktop](https://github.com/micky178527sm-byte/blackhole-desktop)

*macOSデスクトップ上に重力レンズ効果をリアルタイムに描画するアプリケーション*

- Features multi-display support, dynamic UI configuration, and background daemon execution.  
  *マルチモニター対応、動的なUI設定画面、バックグラウンドでのデーモン実行などの機能を実装*
